# 项目简介
一个基于 React 的游戏角色（沈星回）交互式展示页，包含首页，角色介绍，特色玩法等模块。

# 技术栈
## 核心前端框架
- **React+Ts**
## 路由管理：ReactRouter
## UI组件
- **Swiper+ReactTooltip**
## 构建与工程化
- **CRA+pnpm+Webpack+Craco**
--------------------------------------
# 快速开始
## 环境要求
- Node.js 16.0 或更高版本
- pnpm 7.0 或更高版本
## 安装依赖
```bash
pnpm install
``` 
## 本地启动
```bash
 pnpm start
 ```
## 构件生产版本
 ```bash
 pnpm build
 ```
